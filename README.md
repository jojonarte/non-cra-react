# Non CRA React Template

Barebones React template to get you up and running.

## Installation

- Use this template (will generate repository for you)
```
$ git clone <generated repo>
$ cd <cloned folder path>
$ npm install
```

## Running development
```
$ npm run dev
```

## Running production
```
$ npm run prod
```
